My procedure to set up for a HTML5mod (remove the “#” in the URL):

  1. Add the following code in index.html:
    <base href="/">
    2. <div>
  2. Add the following code in main.controller.js:
    $locationProvider.html5Mode(true);
  3. [important] Add the following code after the router in server.js to solve the rewrite problems on server side:
    app.use(function(req, res) {
    res.sendfile(__dirname + '/Public/index.html');
});
    Reference: https://stackoverflow.com/questions/16569841/reloading-the-page-gives-wrong-get-request-with-angularjs-html5-mode/16570533#16570533
