My note about CSS:

  1. The position property specifies the type of positioning method used for an element.
    There are four different position values:

    • static
    • relative
    • fixed
    • absolute
    Click here to learn more.

  2. Use top: 50% / left: 50% in combination with translateX(-50%) translateY(-50%) to dynamically vertically/horizontally center the element.

    position: absolute;
top: 50%;
left: 50%;
transform: translateX(-50%) translateY(-50%);

  3. Draw the "menu icon" with CSS:
    span {
    height: 3px;
    background: #97ff77;
    font-size: 0;
    margin-top: -2px;
    transition: all 1s ease;
    user-select: none;
    width: 100%;
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translateX(-50%) translateY(-50%);
    
    &::before {
        background: #97ff77;                
        position: absolute;
        transform: translateY(-250%);
        height: 100%;
        width: 100%;
        content: '';
    }
    
    &::after {
        background: #97ff77;                
        position: absolute;
        transform: translateY(250%);
        height: 100%;
        width: 100%;
        content: '';
    }
}
  4. What is the difference between visibility:hidden and display:none?
    display:none will not be available in the page and does not occupy any space.
    visibility:hidden hides an element, but it will still take up the same space as before. The element will be hidden, but still affect the layout. visibility:hidden preserve the space, whereas display:none doesn't preserve the space.
