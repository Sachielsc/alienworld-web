Node.js study note

Important Concepts:

  • Node.js is an open source server framework.
  • Node.js allows you to run JavaScript on the server.
  • Node.js is single-threaded but asynchronous.

Learning Progress:

  1. Different useful built-in modules:
    http, fs, URL, formidable, nodemailer
  2. Node.js NPM
  3. Node.js events
  4. Node.js MySQL
    Core tech: Query -- Use SQL statements to read from (or write to) a MySQL database. This is also called "to query" the database.
