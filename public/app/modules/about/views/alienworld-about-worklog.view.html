Where is the problem:
  • The nav bar / title bar is designed to be perfectly merged into the background when user firstly enter the site;
  • After the user scroll down a bit, the title bar will become partially transparent and get a grey background to distinguish it from the main body;
  • However, the white font â€“ grey background is still easily mixing up with the text when user try to read the content of the body or try to watch embedded videos. The partially transparent title bar still jeopardizes the integrity of main body;
  • Users might click on the title bar by mistake when they intend to click on the main body.
My suggestion:
  1. Hide the nav bar in areas where users are trying to read the main body text;
  2. Display the nav bar when users are at the top of EACH big picture of different chapters instead of merely at the top of the first chapter. This will make the nav bar merge into each chapter perfectly and consistently;
  3. Do not display the nav bar with a gray background because it defeats the purpose of using a auto-merged design.
My solution:

CSS:

.hidethis {
        display:none;
        -webkit-transition: all 1s ease;
           -moz-transition: all 1s ease;
            -ms-transition: all 1s ease;
             -o-transition: all 1s ease;
                transition: all 1s ease;
    }

jQuery (Add to index.html)

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>
    <script>
    $(document).ready(function(){
        
        var mainbottom = $('#yui_3_17_2_1_1501347938135_114').offset().top + 0.5 * $('#yui_3_17_2_1_1501347938135_114').height();
        var exporttop = $('#yui_3_17_2_1_1501347938135_311').offset().top;
        var exportbottom = $('#yui_3_17_2_1_1501347938135_311').offset().top + 0.5 * $('#yui_3_17_2_1_1501347938135_311').height();
        var tourismtop = $('#yui_3_17_2_1_1501347938135_352').offset().top;
        var tourismbottom = $('#yui_3_17_2_1_1501347938135_352').offset().top + 0.5 * $('#yui_3_17_2_1_1501347938135_352').height();
        var educationtop = $('#yui_3_17_2_1_1501347938135_431').offset().top;
        var educationbottom = $('#yui_3_17_2_1_1501347938135_431').offset().top + 0.5 * $('#yui_3_17_2_1_1501347938135_431').height();

        $(window).on('scroll',function(){
            scrollposition = Math.round($(window).scrollTop());
            if (scrollposition < mainbottom) {
                $('.sqs-announcement-bar-dropzone').addClass('hidethis');
            } else {
                if (scrollposition > exporttop && scrollposition < exportbottom) {
                    $('.sqs-announcement-bar-dropzone').addClass('hidethis');
                } else {
                    if (scrollposition > tourismtop && scrollposition < tourismbottom) {
                        $('.sqs-announcement-bar-dropzone').addClass('hidethis');
                    } else {
                        if (scrollposition > educationtop && scrollposition < educationbottom) {
                            $('.sqs-announcement-bar-dropzone').addClass('hidethis');
                        } else {
                            $('.sqs-announcement-bar-dropzone').removeClass('hidethis');
                        }
                    }
                }
            }
        });
    });
    </script>
